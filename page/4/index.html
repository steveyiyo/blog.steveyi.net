在官方不支援 macOS Catalina 的裝置安裝 macOS Catalina

macOS 重灌不求人！使用內建回復功能輕鬆重灌 Mac 電腦

很多時候很多種原因我們需要重灌作業系統，比如說系統裝了太多東西覺得變慢 懶得一個一個刪掉、或是換了新的磁碟機、升級Beta版想要降回來穩定發行版、或是系統升級過程中意外斷電等原因。Mac內建了一個好用的工具程式可以協助我們重灌。 開始教學囉 首先請先將電腦開機，在蘋果圖像出現前按住command + R ，如果有韌體密碼會在此步驟要求解鎖。 電腦會先嘗試使用內建的復原磁區開機，如果沒有找到復原磁區的話會使用Internet回復。 Internet回復會使用Wi-Fi或乙太網路，如果找不到可用的網路就會出現驚嘆號的地球符號。 *如果使用的第三方鍵盤找不到command鍵，請把Windows符號鍵當作command使用。 螢幕快照擷取自Apple官網 如果沒有出現上圖畫面的話，接下來會照常以蘋果圖像開機 開機完成後會看到可以選擇語言 選完語言後會看到macOS工具程式 如果是因為系統升級失敗(例如升級過程意外拔到插頭等)、刪錯kext驅動，請跳過磁碟工具程式的部分，直接前往第三頁，這會保留系統卷宗的資料和應用程式。 如果是想重灌（不保留系統卷宗資料）或是更換新的磁碟機，請繼續前往第二頁。 接下來要使用磁碟工具程式清除資料 選擇系統的啟動磁碟或卷宗，命名並選擇格式然後按下清除 如果有看到APFS格式選項（10.13以上才有)，建議選擇APFS。 沒有的話就選Mac OS 擴充格式。 完成後就可以把這視窗關掉，回到macOS工具程式了 接下來就是重新安裝系統的時候了，請選擇重新安裝macOS 接下來請按「繼續」，然後會看到使用條款。 不同意使用條款就不能安裝。 然後請選擇安裝於剛剛清除的磁區或卷宗 接下來就等候安裝吧！ 安裝完會有基本設定，本篇就不贅述了，祝大家使用愉快～

幫 Spotify 或其他音樂軟體加上實時歌詞的方法 (Android)

上次有一篇介紹各家串流平台的免費會員比較，各位可以發現實時歌詞很多家平台都有 但Spotify根據筆者測試只有Premium用戶能用或是有些平台沒有歌詞服務，稍嫌可惜 這時想要聽著Spotify或其他沒歌詞的平台的免費音樂，但又覺得沒歌詞不能一起唱很麻煩嗎 這時候照這篇文章做就可以解決這個問題囉 這篇文章適用Android用戶，iOS/iPadOS用戶在另外一篇唷 注意：Musixmatch for Android的原理是讀取通知，所以有些中文的專輯版音樂被Youtube Music等平台轉為英文名時，可能產生誤判情形，此時就只能改用其他平台或是關掉Floating Lyrics了，如果是MV或是影片大概就辨識不到了 如果使用小米/Redmi/黑鯊/POCO手機的用戶，請將應用程式從最近應用上鎖，並將省電策略設為無限制，可以參考這邊 開始吧 首先請先下載Musixmatch 下載完成後請打開並註冊或登入Musixmatch的帳號 如果有詢問是否加入Musixmatch Premium會員，可以關掉 接下來請允許所需的權限 接下來視需要，登入Spotify或是音樂庫 此處以Spotify示範 輸入帳號密碼 然後滑到最底部按綠色的「同意」按鈕 在Spotify或其他平台隨便播放一首歌，如果跳出這視窗，請按「別再顯示」 接下來會跳出Floating Lyrics的實時歌詞視窗 如果支援實時歌詞的歌曲就會自己跳，如果只支援靜態歌詞就需要自己用手滑，不支援歌詞的話也會出現在這邊 按下Floating Lyrics視窗裡，歌名右邊的圓形按鈕可以更改樣式 比如說更改顏色或是展開歌詞 按下圓形按鈕右邊的三個點按鈕可以看到選單有更多功能 點選「在App內顯示」就可以整個畫面充滿歌詞 打開通知中心可以看到隱藏Floating Lyrics的按鈕 接下來可以測試其他應用程式（但不是每款音樂App都支援） 筆者在Friday音樂隨便播放一首歌，程式本身沒有這首歌的歌詞，此時Musixmatch便非常實用 在Youtube Music測試，不少專輯版音樂都能用，但如前面所述，部分中文歌會產生誤判的情形（我也不知道到底為什麼中文歌名要寫英文） 舉例來說，以下這首歌曲是鄧紫棋的「好想好想妳」 但歌詞卻誤判為另一首(我完全沒聽過的)歌曲，這時我只好關掉Floating Lyrics，或是改用選單的歌詞辨識功能
